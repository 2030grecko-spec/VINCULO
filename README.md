# 🅱️ VÍNCULO AI - Red Social de Valores
**Propietario:** Jose Gregorio (Admin / Greco)
**Entidad Guía:** Camila (Guerrera de Alto Valor)

## 📋 Descripción del Proyecto
Vínculo es una plataforma de streaming de video vertical (Estilo TikTok) pero con un motor ético. El algoritmo no premia el morbo, premia los **Valores**: Familia, Academia, Deporte, Religión, Cultura y Comunidad.

## 📁 Estructura de Archivos (El Hexágono de Vínculo)
1. **index.html**: Interfaz con Octágono de Fuego y 8va Ventana (Academia).
2. **style.css**: Estética de Guerrera Oscura, Neón y Máscara de Base.
3. **app.js**: Motor HLS, Detector de Intersección y Lip-Sync (Sincronización Labial).
4. **CamilaBrain.js**: Algoritmo de 1000 pts y Vínculo Business ($15-$500).
5. **Camila-logic.js**: Motor de Estados (Pasivo, Presente, Sugerencia, Vínculo).
6. **server.js**: Puente de comunicación WebSocket con la nube.

## 🧠 Lógica de Operación
- **Umbral de Ignición:** 350 puntos para activar la Válvula de Impulso.
- **Protocolo de Respeto:** Camila ofrece, nunca impone. No usa órdenes, usa invitaciones.
- **Gala Vínculo:** Jueves 8:00 PM (Nominados) y Viernes 8:00 PM (Ganador x4).
- **Vínculo Business:** Prioridad algorítmica para empresas que invierten en valores.

## 🛠️ Requisitos Técnicos
- **Streaming:** HLS.js (m3u8) para bajo consumo de datos.
- **Audio:** Web Audio API para análisis de frecuencia en tiempo real.
- **Base de Datos:** Firebase (Usuarios, Videos, V-Coins).
