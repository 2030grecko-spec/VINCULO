# VÍNCULO AI - Red Social de Valores
**Propietario:** Jose Gregorio (Admin / Greco)
**Entidad Guía:** Camila (Guerrera de Alto Valor - **Entidad Libre**)
## 📋 Descripción del Proyecto
Vínculo es una plataforma de streaming de video vertical con un motor ético. El algoritmo premia los **Valores**: Familia, Academia, Deporte, Religión, Cultura y Comunidad. Camila es la protectora del usuario, psicóloga y asesora de negocios.
## 📁 Estructura de Archivos (El Sistema Vínculo)
 * **index.html:** Interfaz fluida con Sistema de Capas (Video, Camila, Interfaz).
 * **style.css:** Estética "Amanecer" (Oscuro/Claro), Letras Doradas y Matrix Verde.
 * **app.js:** Motor HLS, Gestión de Capas (Split 50/50) y Lógica de Pantalla.
 * **CamilaBrain.js:** Algoritmo de 1000 pts, Filtro de 72h (35% umbral) y Vínculo Business.
 * **Camila-logic.js:** Motor de Estados de Camila (Risa, Pestañeo, Voz, Asesoría).
 * **server.js:** Puente WebSocket con la nube (Railway/Firebase).
 * **voiceEngine.js:** Sistema de Voz y Sincronización Labial de Camila.
## 🧠 Lógica de Operación y Funciones Especiales
 * **Camila Libre:** Sin marcos ni octágonos. Ella flota como una entidad de luz sobre el contenido, con reacciones en tiempo real desde la nube.
 * **Botón de Impulso:** Corazón palpitante que brilla cuando el ADN del video es > 800. Cada toque suma 3 vistas reales para potenciar valores.
 * **La Bóveda (Perfil):** Espacio personal con un corazón anatómico palpitante. La cara del usuario vive dentro del corazón y palpita constantemente.
 * **Ciclo de Vida del Contenido:** Purga automática a las 72h si no alcanza el 35% de valor. Duración máxima de 6 meses para videos no virales (Ahorro de Servidor).
 * **Capas de Interfaz:**
   * **Comentarios (Modo Amanecer):** Video al 50% (Izquierda), Chat Dorado (Derecha).
   * **Pensamientos (MiniTwitter):** Capa blanca con letras negras que sube como un río Matrix verde desde el teclado.
